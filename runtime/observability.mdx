Hyperterse integrates [OpenTelemetry](https://opentelemetry.io) for distributed tracing and metrics export via OTLP (OpenTelemetry Protocol).

## Architecture

```
┌──────────────┐
│  Hyperterse  │
│   Runtime    │
└──────┬───────┘
       │
       ├─> Traces (spans)
       │     │
       │     └─> OTLP/gRPC → Collector/Backend
       │
       └─> Metrics (counters/histograms)
             │
             └─> OTLP/gRPC → Collector/Backend
```

**Supported Backends:**
- Grafana Cloud
- Honeycomb
- Datadog
- New Relic
- Self-hosted OpenTelemetry Collector

## Configuration

### Environment Variables

```bash
# Enable observability (default: false)
export HYPERTERSE_OTEL_ENABLED=true

# OTLP endpoint (default: localhost:4317)
export HYPERTERSE_OTEL_ENDPOINT=otel-collector.example.com:4317

# Service metadata
export HYPERTERSE_OTEL_SERVICE_NAME=hyperterse-api
export HYPERTERSE_OTEL_SERVICE_VERSION=1.0.0
export HYPERTERSE_OTEL_ENVIRONMENT=production

# Trace sampling (default: 1.0 = 100%)
export HYPERTERSE_OTEL_TRACE_SAMPLING_RATIO=0.1  # Sample 10%

# Protocol (default: grpc)
export HYPERTERSE_OTEL_PROTOCOL=grpc

# Enable/disable signals (default: all enabled)
export HYPERTERSE_OTEL_TRACES_ENABLED=true
export HYPERTERSE_OTEL_METRICS_ENABLED=true
export HYPERTERSE_OTEL_LOGS_ENABLED=true  # Reserved for future use
```

**Implementation:** `core/observability/config.go:28-52`

## Initialization

Observability providers are initialized during runtime startup:

```go
obsProviders, err := observability.Setup(context.Background(), model, serviceVersion)
if err != nil {
    return nil, fmt.Errorf("failed to initialize observability: %w", err)
}
```

**Source:** `core/runtime/server/server.go:57-60`

### Setup Flow

1. **Resolve Configuration** (`config.go:28-84`)
   - Load defaults
   - Apply environment variable overrides
   - Substitute `{{ env.* }}` placeholders
   - Validate sampling ratio (0.0-1.0)

2. **Build Trace Provider** (`tracing.go:13-46`)
   - Create OTLP gRPC exporter
   - Configure resource attributes (service name, version, environment)
   - Set up sampler (trace ID ratio-based)
   - Attach batch span processor

3. **Build Meter Provider** (`metrics.go:31-63`)
   - Create OTLP gRPC exporter
   - Configure resource attributes
   - Attach periodic reader (default: 60s interval)

4. **Register Global Providers**
   ```go
   otel.SetTracerProvider(traceProvider)
   otel.SetMeterProvider(meterProvider)
   ```
   **Source:** `setup.go:57-58`

5. **Set Error Handler**
   ```go
   otel.SetErrorHandler(otelLoggerErrorHandler{log: logger.New("observability")})
   ```
   **Source:** `setup.go:59`

   Routes OpenTelemetry internal warnings to Hyperterse logger.

### HTTP Handler Spans

```go
ctx, span := rt.tracer.Start(req.Context(), "http."+endpoint)
span.SetAttributes(
    attribute.String("http.request.method", req.Method),
    attribute.String("http.route", endpoint),
)
defer span.End()
```

**Source:** `server.go:352-357`

**Attributes:**
- `http.request.method`: HTTP method (GET, POST, DELETE)
- `http.route`: Endpoint path (`/mcp`, `/heartbeat`)
- `http.response.status_code`: HTTP status code (added after handler)

**Error Handling:**
```go
if recorder.statusCode >= 500 {
    span.SetStatus(codes.Error, "server_error")
}
```

**Source:** `server.go:365-367`

### Executor Spans

```go
defer span.End()
```

**Source:** `executor.go:41-44`

**Attributes:**
- `tool.name`: Tool identifier

```go
if err != nil {
    span.SetStatus(codes.Error, "tool_execution_failed")
```

**Source:** `executor.go:146`

### Sampling

Trace sampling is configured via `HYPERTERSE_OTEL_TRACE_SAMPLING_RATIO`:

```go
sdktrace.WithSampler(sdktrace.TraceIDRatioBased(cfg.TraceSamplingRate))
```

**Source:** `tracing.go:41`

**Sampling Strategies:**
- **1.0** (100%): Development, debugging
- **0.1** (10%): Low-traffic production
- **0.01** (1%): High-traffic production

**Note:** Sampling decisions propagate to child spans (same trace ID = same decision).

## OTLP Export

### Trace Export

```go
exporter, err := otlptracegrpc.New(
    ctx,
    otlptracegrpc.WithEndpoint(cfg.OTLPEndpoint),
    otlptracegrpc.WithInsecure(),
)
```

**Source:** `tracing.go:18-22`

**Transport:** gRPC (insecure by default)

**Batch Processor:**
```go
sdktrace.WithBatcher(exporter)
```

Spans are buffered and exported in batches (default: 512 spans or 5s).

### Metric Export

```go
exporter, err := otlpmetricgrpc.New(
    ctx,
    otlpmetricgrpc.WithEndpoint(cfg.OTLPEndpoint),
    otlpmetricgrpc.WithInsecure(),
)
```

**Source:** `metrics.go:36-40`

**Periodic Reader:**
```go
sdkmetric.WithReader(
    sdkmetric.NewPeriodicReader(exporter),
)
```

Metrics are exported every 60 seconds (default interval).

## Shutdown

Providers are gracefully shut down during runtime stop:

```go
if r.observability != nil {
    shutdownCtx, shutdownCancel := context.WithTimeout(context.Background(), 10*time.Second)
    defer shutdownCancel()
    if err := r.observability.Shutdown(shutdownCtx); err != nil {
        log.Warnf("Observability shutdown error: %v", err)
    }
}
```

**Source:** `server.go:286-292`

**Shutdown Flow:**
1. Flush remaining spans from batch processor
2. Export final metrics
3. Close OTLP exporters

**Timeout:** 10 seconds

## Example: Self-Hosted Collector

**docker-compose.yml:**
```yaml
services:
  otel-collector:
    image: otel/opentelemetry-collector:latest
    command: ["--config=/etc/otel-collector-config.yaml"]
    volumes:
      - ./otel-collector-config.yaml:/etc/otel-collector-config.yaml
    ports:
      - 4317:4317  # OTLP gRPC
      - 4318:4318  # OTLP HTTP

  hyperterse:
    environment:
      - HYPERTERSE_OTEL_ENABLED=true
      - HYPERTERSE_OTEL_ENDPOINT=otel-collector:4317
```

**otel-collector-config.yaml:**
```yaml
receivers:
  otlp:
    protocols:
      grpc:
        endpoint: 0.0.0.0:4317

processors:
  batch:

exporters:
  prometheus:
    endpoint: 0.0.0.0:8889
  jaeger:
    endpoint: jaeger:14250

service:
  pipelines:
    traces:
      receivers: [otlp]
      processors: [batch]
      exporters: [jaeger]
    metrics:
      receivers: [otlp]
      processors: [batch]
      exporters: [prometheus]
```

## Security: Attribute Redaction

Sensitive attribute values are automatically redacted:

```go
func RedactAttributeValue(key string, value string) string {
    lower := strings.ToLower(key)
    for _, needle := range secretKeySubstrings {
        if strings.Contains(lower, needle) {
            return "[REDACTED]"
        }
    }
    return value
}
```

**Source:** `contract.go:39-47`

**Redacted Substrings:**
- password
- passwd
- secret
- token
- api_key
- apikey
- authorization
- connection_string
- dsn

**Example:**
```go
RedactAttributeValue("database_password", "secret123")
// → "[REDACTED]"
```
